# CreatorOps Voucher Generator

> Transform creator voucher workbooks into ready-to-send outreach messages in seconds.

A browser-native workflow automation tool that transforms repetitive creator operations into a fast, standardised and scalable process through intelligent Excel parsing and rule-based automation.

Originally inspired by real creator operations workflows in a TikTok Shop agency, this project generalises those operational challenges into a reusable workflow automation product that can be adapted by creator marketing and social commerce teams.
